About


My name is Bernard Oosthuizen and I am a full-stack web developer. I design, build and deploy custom web applications. I create dynamic and responsive websites. I have been working with web technologies for 3 years. I also provide hosting solutions.

When dealing with clients I pay attention to detail, I communicate and I make sure all the users' needs are met.

For the front-end I use the following technologies: HTML, CSS, Javascript, Node.js, Vue.js, SASS, Gulp and Webpack. And for the back-end I use the following: Python, Django, SQL and PostgreSQL.

I live near Cape Town, South Africa. When I was 15 I started teaching myself web development. In my free time I enjoy mountain biking, running, fishing and playing tennis.
